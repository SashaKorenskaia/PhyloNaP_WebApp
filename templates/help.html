PhyloNaP Help Center

About PhyloNaP

Natural products (NPs), many of which act as antibiotics, are small molecules synthesized by living organisms. With the rise of antibiotic resistance, discovering new antibiotics is essential. Genome mining is a powerful method that explores an organism's biosynthetic potential based on its genetic sequence, identifying biosynthetic gene clusters (BGCs) involved in the production of novel antibiotics.

PhyloNaP is a tool designed to facilitate the prediction of tailoring enzyme functions, enhancing our ability to discover new antibiotics crucial in combating antibiotic resistance.

1. Database Content Overview

Data Sources & Accumulation

PhyloNaP's database contains phylogenetic trees for various enzyme families involved in natural product biosynthesis. The data is accumulated from:

  • Curated datasets: Manually reviewed enzyme families with experimental validation
  • Automated collections: Computationally generated trees from sequence databases
  • Literature mining: Enzyme functions extracted from published research
Database content visualization

Automatic Generation Pipeline

  1. Sequence Collection: Gather homologous sequences from public databases
  2. Multiple Sequence Alignment: Align sequences using MAFFT/MUSCLE
  3. Phylogenetic Reconstruction: Build trees using maximum likelihood methods
  4. Annotation Integration: Add functional annotations and metadata
  5. Quality Control: Validate tree topology and bootstrap support

Understanding Result Pages

Results Table
  • Query: Your input sequence identifier
  • Confidence: Placement reliability (0-1)
  • Branch Length: Evolutionary distance
  • Family/Superfamily: Functional classification
Tree Placement View
  • Query Position: Shows where your sequence fits
  • Colored Branches: Highlight placement confidence
  • Annotations: Display functional context
  • Interactive Exploration: Navigate and analyze
Interpreting Results
Confidence Levels:
  • 0.8-1.0: High confidence placement
  • 0.5-0.8: Moderate confidence
  • <0.5: Low confidence, interpret carefully
Branch Length:
  • <0.5: Close homolog, reliable function
  • 0.5-1.0: Moderate distance
  • >1.0: Distant relationship Caution

4. Submission Portal

How to Submit Sequences

  1. Prepare Your Data:
    • Protein sequences in FASTA format
    • One or multiple sequences per submission
    • Include descriptive sequence names
  2. Upload Options:
    • Paste sequences directly into text area
    • Upload FASTA file from your computer
    • Maximum file size: 10MB
  3. Job Configuration:
    • Provide email for notification (optional)
    • Set job description for easy identification
    • Choose analysis parameters if needed
  4. Submit & Track:
    • Receive unique job ID for tracking
    • Monitor progress on results page
    • Results typically ready in 5-30 minutes
